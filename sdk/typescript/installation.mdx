## Basic Setup

```typescript
import { Raindrop } from 'rd-mini';

const raindrop = new Raindrop({
  apiKey: process.env.RAINDROP_API_KEY,
});
```

## Configuration Options

```typescript
const raindrop = new Raindrop({
  // Required
  apiKey: process.env.RAINDROP_API_KEY,

  // Optional
  baseUrl: 'https://api.raindrop.ai',  // Custom API endpoint
  debug: true,                          // Log all events to console
  disabled: process.env.NODE_ENV === 'test',  // Disable in tests

  // Advanced (tune for production)
  flushInterval: 1000,   // ms between auto-flushes (default: 1000)
  maxQueueSize: 100,     // Flush when queue reaches this size (default: 100)
  maxRetries: 3,         // Retry failed requests (default: 3)
});
```

## Supported Providers

rd-mini supports wrapping these AI clients:

| Provider | Import | Wrap |
|----------|--------|------|
| OpenAI | `import OpenAI from 'openai'` | `raindrop.wrap(new OpenAI())` |
| Anthropic | `import Anthropic from '@anthropic-ai/sdk'` | `raindrop.wrap(new Anthropic())` |
| Vercel AI SDK | `import { openai } from '@ai-sdk/openai'` | `raindrop.wrap(openai('gpt-4o'))` |
