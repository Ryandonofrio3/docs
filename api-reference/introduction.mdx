# HTTP API

The rd-mini SDKs use these HTTP endpoints under the hood. You can also call them directly.

## Base URL

```
https://api.raindrop.ai/v1
```

## Authentication

All requests require a Bearer token:

```bash
curl https://api.raindrop.ai/v1/events/track \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '[...]'
```

## Rate Limits

- 1000 requests per minute per API key
- 1 MB max payload size per request
- Batch up to 100 events per request
